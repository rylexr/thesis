Actividades

Se analizará el material existente y se procederá a documentar todo el trabajo relacionado con reconstrucción tridimensional, con tal de determinar una base comprensiva de información para el sistema.

Se procederá a evaluar técnicas existentes para cada una de las etapas de la reconstrucción propuestas en este documento (calibración, profundidad, modelado), con tal de determinar algoritmos que maximicen rapidez y calidad de reconstrucción. También se procederá a evaluar el rendimiento (cuadros por segundo) de diferentes cámaras web para determinar cómo afecta al sistema.

Finalmente, se procederá a la evaluación de diferentes bibliotecas de código abierto con las cuales realizar la construcción del sistema.
